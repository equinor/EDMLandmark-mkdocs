Integrated EDM™ Workflows

The following workflows offer suggestions for using Landmark software for everyday business activities in the Drilling and Completions industry. Click on the following links to access these workflows.

Well Planning

A typical workflow that might be used while planning a well. This workflow is not the only possible workflow that could be used during the planning phase, but rather is representative of a possible scenario.

Survey Sharing

A typical workflow that can be used to compare planned and actual trajectories.

Drilling Planning to Reporting

Typical workflows that begin in PROFILE™, StressCheck™, or WELLPLAN™ software, and end using the OpenWells® application. In all of these workflows, drilling operation planning data is available to OpenWells after it was input using PROFILE, StressCheck, or WELLPLAN software.

Drilling Operations to Analysis

Typical workflows that begin in OpenWells® software and end using WELLPLAN™ software. In all of these workflows, the drilling operation report data is input into OpenWells software, and then the data is analyzed using the WELLPLAN application.
